# Polarimetric SAR Image Segmentation Based on Non-local and Region-level Conditional Random Field Model with Edge Preserved

**Python implementation of the NRLCRF-EP model for polarimetric SAR image segmentation, addressing misclassified isolated pixels, multi-scale adaptability, and edge detail preservation.**

**Submitted to:** JSTARS

**Authors:** Sutong Wang, Chun Liu, and Jian Yang
